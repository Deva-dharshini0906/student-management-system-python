# Student Management System

A console-based Student Management System developed using Python.

## Features
- Add Student
- View Students
- Search Student
- Delete Student

## Technologies Used
- Python
- Lists
- Dictionaries

## Project Description
This application helps manage student records through a simple command-line interface. Users can add, view, search, and delete student information efficiently.
